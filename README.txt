Add a WordPress popup box to the admin page settings, including backend options.

== Installation ==
* Install the plugin from the \\\'Plugins\\\' section in your dashboard (Go to `Plugins > Add New > Search` and search for WP Popup Box).
* Activate it through the \\\'Plugins\\\' section.
